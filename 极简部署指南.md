## 第 2 步：告诉前端后端在哪（1 分钟）

把下面这段代码复制到文件 `src/providers/trpc.tsx`，**覆盖原来内容**：

```tsx
import { createTRPCReact } from "@trpc/react-query";
import { httpBatchLink } from "@trpc/client";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import superjson from "superjson";
import type { AppRouter } from "../../api/router";
import type { ReactNode } from "react";

export const trpc = createTRPCReact<AppRouter>();

const queryClient = new QueryClient();

// ======= 改成你的 Render 网址 =======
const API_URL = "https://private-desktop.onrender.com/api/trpc";
// =====================================

const trpcClient = trpc.createClient({
  links: [
    httpBatchLink({
      url: API_URL,
      transformer: superjson,
      fetch(input, init) {
        return globalThis.fetch(input, {
          ...(init ?? {}),
          credentials: "include",
        });
      },
    }),
  ],
});

export function TRPCProvider({ children }: { children: ReactNode }) {
  return (
    <trpc.Provider client={trpcClient} queryClient={queryClient}>
      <QueryClientProvider client={queryClient}>
        {children}
      </QueryClientProvider>
    </trpc.Provider>
  );
}
```

> 注意：把 `https://private-desktop.onrender.com` 换成你**第 1 步实际拿到的网址**。

然后提交代码：

```bash
git add src/providers/trpc.tsx
git commit -m "update api url"
git push origin main
```

Render 会自动重新部署（等 2 分钟）。

## 第 3 步：改登录回调地址（1 分钟）

1. 打开 https://platform.kimi.com
2. 找到你的应用（ID 结尾是 `a899c295`）
3. 把 **OAuth 回调地址**改成：
   ```
   https://你的-render-网址/api/oauth/callback
   ```
   例如：`https://private-desktop.onrender.com/api/oauth/callback`
4. 点保存

## 完成

打开你的 Render 网址，应该能看到登录页。用 Kimi 账号登录后：

- 数据自动同步到云端
- 换任何设备打开同一个网址，数据都在
- 状态栏会显示"云端同步"小云朵图标

---

## 如果出问题

**页面显示 API 不可用？**
→ 检查 `src/providers/trpc.tsx` 里的 `API_URL` 是否是你的 Render 网址。

**登录后跳回空白页？**
→ 检查 Kimi 平台的回调地址是否是你的 Render 网址 + `/api/oauth/callback`。

**渲染日志报错？**
→ 打开 Render 面板 -> Logs，截图发给我。

**免费额度够用吗？**
→ Render 免费版每月 750 小时运行时间，个人使用完全够。缺点是 15 分钟不用会自动休眠，下次打开需等 30 秒唤醒。
